プレイ側の信号（何が送られる？）

送信元信号送信先結果
プレイヤー掲示板投稿Nostr全端末に新着表示
プレイヤーコメント削除Nostr全端末で削除反映
プレイヤープレイログWorker APID1に監査保存
プレイヤー何人目取得Worker APIタイトル下に番号表示
プレイヤー手紙確認チェックWorker API確認人数に加算
やさしい説明

プレイヤーの操作は「みんなに見せる情報（Nostr）」と「記録する情報（Worker）」に分けて送られます。

投稿/削除は Nostr イベント、監査や番号/確認チェックは REST API で Worker に送信されます。

管理画面監査ログ取得Worker APID1のログを表示
管理画面手紙確認人数取得Worker API現在人数を表示

  1. このゲームは「表示ページ」「リアルタイム共有」「記録API」の3層に分けて運用しています。
